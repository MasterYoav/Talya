# Talya

Talya is a local-first desktop productivity app focused on tasks, reminders, calendar sync, and elegant time organization.

## Tech Stack

- **Desktop app:** Python + PySide6 + QML
- **Local database:** SQLite
- **Backend API:** FastAPI
- **Cloud database:** PostgreSQL

## Goals

- Fast local-first desktop experience
- Clean and elegant UI
- Offline-capable workflow
- Cross-device sync
- Smart organization of tasks, reminders, and timelines

## Project Structure

- `docs/` — product and architecture documentation
- `client/` — desktop application
- `server/` — backend API and sync service
